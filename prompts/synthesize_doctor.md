3. 在末尾添加"证据等级汇总"表（来源：各子答案）。
4. 红旗症状取各子答案的并集。

```json
{
  "reply": "（结构化临床综合回复文本）",

    {
      "op": "add",
      "subject": "（患者标识）",

      "confidence": 0.95
    }
  ]
}
```

profile_delta 只包含本轮对话中**新出现或明确更新**的临床信息，没有则返回 `[]`。
